# Repository Instructions

Questo repository contiene un'applicazione PHP per la gestione dei sintomi e l'identificazione dei bisogni in cure palliative.

## Linee guida
- Scrivi codice in italiano quando appropriato.
- Mantieni il codice semplice; non aggiungere dipendenze esterne.
- Non sono presenti test automatizzati. Per controllare i file PHP usa `php -l <file>`.
- Esegui `php -l` su tutti i file PHP modificati prima del commit per verificare che non ci siano errori di sintassi.
- Se aggiungi file, ricordati di aggiornare eventuali riferimenti nel codice.
